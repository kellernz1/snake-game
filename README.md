# 🐍 Snake Game Web App

A modern and interactive **Snake Game web application** built with **HTML, CSS, and JavaScript**.  
This project features a clean **dark theme UI**, smooth animations, and responsive controls for both keyboard and on-screen buttons.

---

## 🌐 Live Demo

👉 [https://kellernz1.github.io/snake-game](https://kellernz1.github.io/snake-game)

---

## 🚀 Features

* 🎯 **Precise Snake Mechanics**  
  The snake moves smoothly and only collides when it hits walls or itself.

* 🎨 **Modern Dark Theme**  
  Clean layout with shadows, rounded snake segments, and a stylish game board.

* 🐍 **Fancy Rainbow Snake**  
  Each segment has a rainbow gradient for a visually appealing look.

* 🍎 **Shiny Apple**  
  Apples are drawn with a glowing effect and **never spawn inside the snake**.

* ⬆️⬇️⬅️➡️ **Keyboard & On-Screen Controls**  
  Play with arrow keys or click the buttons on-screen; rapid key presses are handled safely.

* 📝 **Scoreboard**  
  Tracks your current score and updates live during gameplay.

* 🔄 **Restart Game**  
  Restart anytime with a single button click.
